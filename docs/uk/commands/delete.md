## Базове використання

```sh
# Конкретний файл
video-manager delete <ім'я-файлу>

# RegExp (видалити всі mp4)
video-manager delete -r '[a-z]+\.mp4'
```

## Опції

### -y, --yes

Пропустити підтвердження видалення.
